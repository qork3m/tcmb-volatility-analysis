# CBRT Rate Decisions and USD/TRY Volatility Analysis

This project analyzes the relationship between CBRT interest rate decisions 
and USD/TRY exchange rate volatility between 2018-2026.

## Findings
In 2018, geopolitical tensions caused a sharp spike in USD/TRY volatility. 
During 2021-2022, the CBRT cut interest rates aggressively despite rising 
inflation, leading to another volatility surge. Following the policy reversal 
in mid-2023, the CBRT raised rates sharply and volatility declined steadily.

## Data Sources
- CBRT interest rate data: EVDS API
- USD/TRY exchange rate: Yahoo Finance API

## Libraries
pandas, matplotlib, evds, yfinance, python-dotenv
